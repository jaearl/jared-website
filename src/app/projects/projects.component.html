SBOL Canvas Web-based CAD for Synthetic Biology

With guidance from Stanford educated Professor Chris Myers, my undergraduate capstone team comprised of Logan Terry, Sam Thayer, and myself built the worlds first open source, web-based, synthetic biology CAD program. Still under active development, SBOL Canvas is being used across the globe to aid in the design and sharing of novel constructs. SBOL Canvas was featured at the Twelfth International Workshop on Bio-Design Automation (IWBDA) in 2020, and is expected to be published in the high-impact journal, ACS Synthetic Biology, in 2021.

OGame API A web browser game hacking prototype


I had a lot of fun hacking a web browser game called OGame, and interfacing a small expert system to the API I wrote for it (I never actually played the game using the bot ;)). I learned a lot about how websites work, how Javascript is parsed, graph algorithms, and the Python programming language. I also learned some lessons about program stability. With how often the website changes, the API was always unstable (and undoubtedly no longer works), but the instability is encapsulated in the OGameAPI interface; so if the API is updated, one can go on developing the bot as if nothing has changed.
